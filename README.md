# Vimo App - Centro de Comando de Operaciones de Seguridad

Es una aplicación complementaria a un proyecto físico. Esta aplicación está desarrollada utilizando Expo y React Native para conectar operadores en vivo y coordinar la seguridad física.

## Comenzar

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:
   ```bash
   npx expo start
   ```

Puedes probar la aplicación en tu celular usando la aplicación móvil **Expo Go**.
